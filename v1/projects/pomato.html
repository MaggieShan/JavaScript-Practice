A productivity application focused around the pomodoro studying technique. Pomato immerses the user in a 25min studying period before providing a game afterwards as a reward. The game itself can only last 5min and is designed to track progess, encouraging players to get through their studying period before playing again. The concept and prototype were made for the hackathon UofTHacks 2019 and was made with Adobe Illustrator, Android Studio and Unity.

My Roles: Creating the graphics for the game, designs for the interface and using java to code the timer features of the app.
